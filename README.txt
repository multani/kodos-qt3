
Kodos is a python GUI for creating, debugging and maintaining 
regular expressions.  Kodos is released under the GNU Public License (GPL).

More information is available at:
http://kodos.sourceforge.net

Kodos was created by Phil Schwartz (phil_schwartz@users.sourceforge.net).



Obtaining Kodos:
================

Download the latest version of Kodos from http://kodos.sourceforge.net

If you downloaded a source distribution, unpack it as appropriate:

Linux/Unix>   tar zxvf kodos-VERSION.tar.gz

Windows>      unzip kodos-VERSION.zip



Building Kodos:
===============

Change into the directory where Kodos was extracted to (typically, kodos-VERSION)

Execute the following command:

Linux/Unix/Windows> python setup.py build




Installing Kodos:
=================

Linux/Unix/Windows> python setup.py install    # you may need to this as root, where applicable
